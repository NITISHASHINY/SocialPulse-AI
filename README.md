# SocialPulse-AI 🚀

**SocialPulse-AI** is a web application developed for the **ET Hackathon 2026**.  
It leverages AI to analyze social media data, providing insights to help users track trends, monitor engagement, and make data-driven decisions.

---

## 🏆 Hackathon Context

- **Event:** ET Hackathon 2026  
- **Project Type:** AI + Social Media Analytics  
- **Team/Author:** NITISHASHINY  

This project was built as a submission for the ET Hackathon, focusing on **real-time social media insights and trend analysis**.

---

## ✨ Features

- Real-time social media analytics  
- AI-driven sentiment analysis  
- Trend detection and visualization  
- User-friendly dashboard  
- Export insights and reports

---

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Python (Flask/FastAPI)  
- **Database:** PostgreSQL / MongoDB  
- **AI/ML:** PyTorch, Transformers  
- **Others:** Docker, REST APIs
